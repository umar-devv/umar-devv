## 🍔 Featured Project: Food Delivery Database

A fully functional NoSQL database system for a food delivery platform (like Foodpanda/UberEats).

### 📊 Project Overview

| Feature | Description |
|---------|-------------|
| **Database** | MongoDB (NoSQL) |
| **Language** | Java 17 |
| **Build Tool** | Maven |
| **GUI Tool** | MongoDB Compass |

### 🗂️ Database Collections

| Collection | Purpose |
|------------|---------|
| **Customers** | User profiles and order history |
| **Restaurants** | Partner restaurant details |
| **MenuItems** | Food items and pricing |
| **Orders** | Customer order tracking |
| **Riders** | Delivery personnel details |
| **Payments** | Transaction records |

### ⚡ Key Features Implemented

- ✅ **CRUD Operations** — Create, Read, Update, Delete records
- ✅ **Aggregation Pipeline** — For generating reports and analytics
- ✅ **Indexing** — Optimized query performance (up to 100x faster)
- ✅ **Java-MongoDB Integration** — Lightweight Java app connecting to MongoDB
